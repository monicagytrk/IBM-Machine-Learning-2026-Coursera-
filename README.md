## About The Course

The IBM Machine Learning Professional Certificate consists of 6 courses that provide solid theoretical understanding and considerable practice of the main algorithms, uses, and best practices related to Machine Learning. The Professional Certificate has a strong emphasis on developing the skills that help advance a career in Machine Learning. All the courses include a series of hands-on labs and final projects.

Below is the list of classes and projects that I submitted for the course: 

1. Exploratory Data Analysis for Machine Learning
    - Project: Utilize data visulatization to gain insight for churn and unchurned Telco customer.
    - Dataset: https://www.kaggle.com/datasets/yeanzc/telco-customer-churn-ibm-dataset
2. Supervised Machine Learning: Regression
    - Project: Compare various combinations of supervised regression models and regularization techniques to identify the most accurate and robust predictive model for house prices in the Boston Housing Dataset.
    - Dataset: https://www.kaggle.com/datasets/meetpatel19aug/datacsv
3. Supervised Machine Learning: Classification
    - Project: Analyze and predict customer payment behavior as a default payer or not using 6 classification models.
    - Dataset: https://archive.ics.uci.edu/dataset/350/default+of+credit+card+clients
4. Unsupervised Machine Learning
    - Project: Create a clustering of a grouped playlist from all spotify listed songs from 1957 to 2020, using various unsupervised learning methods.
    - Dataset: https://www.kaggle.com/datasets/ryanholbrook/fe-course-data/data
5. Deep Learning and Reinforcement Learning
    - Project: Predicting ventilator pressure in a sedated patient simulation using deep learning methodologies.
    - Dataset: https://www.kaggle.com/competitions/ventilator-pressure-prediction/data?select=train.csv
6. Machine Learning Capstone
    - Project: Develop and evaluate four recommender models that suggest the optimal next device model for a user based on the demographic content features and ratings derived from their behavior class.
    - Dataset: https://www.kaggle.com/datasets/valakhorasani/mobile-device-usage-and-user-behavior-dataset
